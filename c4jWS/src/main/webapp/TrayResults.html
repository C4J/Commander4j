Table APP_QM_PANEL_TRAY_RESULTS

Field NameTypeSizePrimary KeyComment
TRAY_IDLong1Link to APP_QM_PANEL_TRAY
SAMPLE_IDLong2Link to APP_QM_TRAY_SAMPLE
USER_IDString203User ID of person recording result
TEST_IDString50Test Identification
VALUEString50Result
UPDATEDTimestampSystem maintained
CREATEDTimestampSystem maintained

This table records a result per user for each sample in the APP_QM_TRAY_SAMPLE table

Transactions

GET (Read)

 Retrieve All Results for a specific trayID and userID
 http://localhost:8080/c4jWS/TrayResults?trayID={id}&userID={id} 

 Retrieve record for specific trayID, sampleID and userID primary key
 http://localhost:8080/c4jWS/TrayResults?trayID={id}&sampleID={id}}&userID={id}

POST (Create)

 Inserts data as per JSON Body
 http://localhost:8080/c4jWS/TrayResults

*Server maintains created/updated record time stamps.

JSON Example - POST

[
  {
    "trayID": 2,
    "sampleID": 830196,
    "userID": "DAVE",
    "testID": "PANEL",
    "value": "052"
  }
]

 Updates data as per JSON Body
 http://localhost:8080/c4jWS/TrayResults

*Server maintains created/updated record time stamps.

JSON Example - PUT

[
  {
    "trayID": 2,

 Deletes Tray as per parameters for trayID,sampleID and userID
 http://localhost:8080/c4jWS/TrayResults?trayID={id}&sampleID={id}&userID={id}

 http://localhost:8080/c4jWS/TrayResults
