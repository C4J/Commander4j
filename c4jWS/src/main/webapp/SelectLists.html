Table APP_QM_SELECTLIST

Field NameTypeSizePrimary KeyComment
SELECT_LIST_IDString401Key Identification.
VALUEString80Value actually stored in database result table and passed in c4jWS/TrayResults
DESCRIPTIONString255Detailed description of value - used in combo box.
SEQUENCEString255Sort Order
VISIBLEString255Determines if an item should appear in the list. Allows and item to be hidden but not deleted.

For this application determine the selectListID key value to use via a REST request to c4jWS/TrayResults using key "QM PANEL SELECTLIST 1"

Transactions

GET (Read)

 Retrieve value for a specified systemKey
 http://localhost:8080/c4jWS/SelectLists?selectListID={key}
