Table APP_QM_PANEL

Field NameTypeSizePrimary KeyComment
PANEL_IDLong1Auto Generated increments for each day or session.
PANEL_DATELongDate and time of the Panel inspection
STATUSString15Panel, Complete, Closed
UPDATEDTimestampSystem maintained
CREATEDTimestampSystem maintained

Transactions

GET (Read)

 Retrieve Panel for a specified panelID
 http://localhost:8080/c4jWS/Panels?panelID={id} 
*ADDITIONAL PARAMS NEEDED FOR DATE RANGE OR MAX RECORDS*

 Retrieve Panel records for specific status
 http://localhost:8080/c4jWS/Panels?status={status} 
*ADDITIONAL PARAMS NEEDED FOR DATE RANGE OR MAX RECORDS*

Status Codes

StatusMeaning
PanelIn progress Panel
CompleteCompleted Panel - not yet posted to results table
ClosedClosed Panel - results posted

JSON Example

[
  {
    "panelID": 1,
    "panelDate": "2022-09-28T11:21:41",
    "status": "Panel",
    "updated": "2022-09-28T11:22:08",
    "created": "2022-09-28T11:22:05"
  }
]

POST (Create)

 Inserts data as per JSON Body
 http://localhost:8080/c4jWS/Panels

*Server maintains created/updated Timestamps - leave panelID blank as the server will auto create and return in response body.

JSON Example - POST

[
 {
    "plant": "Pouch 4a",
    "description": "New Product Trial",
    "status": "Panel"
  }
]

JSON Example - Response

[
 {
    "panelID": 1,
    "trayID": 8,
    "plant": "Pouch 4a",
    "description": "New Product Trial",
    "status": "Prepare",
    "created": "2022-09-25T08:52:30"
  }
 ]

PUT (Update)

 Updates data as per JSON Body
 http://localhost:8080/c4jWS/Panels

*Server maintains created/updated record time stamps.

JSON Example - PUT

[
 {
    "panelID": 1,
    "trayID": 8,
    "plant": "Pouch 4a",
    "description": "New Product Trial",
    "status": "Panel"
  }
]

JSON Example - Response

[
 {
    "panelID": 1,
    "trayID": 8,
    "plant": "Pouch 4a",
    "description": "New Product Trial",
    "status": "Panel",
    "updated": "2022-09-25T18:55:30",
    "created": "2022-09-25T08:52:30"
  }
 ]

DELETE (Delete)

 Deletes Tray as per JSON Body
 http://localhost:8080/c4jWS/Panels

 Deletes Tray as per parameters for panelID and trayID
 http://localhost:8080/c4jWS/Trays?panelID={id}&trayID={8} 

JSON Example - PUT

[
 {
    "panelID": 1,
    "trayID": 8,
  }
]
