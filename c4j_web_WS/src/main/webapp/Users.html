Table APP_QM_USERS

Field NameTypeSizePrimary KeyComment
PANEL_IDLong1Link to Panel record.
USER_IDLong2User ID
CREATEDTimestampSystem maintained

Transactions

GET (Read)

 Retrieve Users for a specified panelID
 http://localhost:8080/c4jWS/Users?panelID={id} 

 Retrieve User by panelID and userID
 http://localhost:8080/c4jWS/Users?panelID={id}&userID=DAVID 

JSON Example

[
  {
    "panelID": 1,
    "userID": "DAVID",
    "created": "2022-09-30T16:43:53"
  }
]

POST (Create)

 Inserts data as per JSON Body
 http://localhost:8080/c4jWS/lUsers

JSON Example - POST

[
  {
    "panelID": 1,
    "userID": "MANDY",
  }
]

JSON Example - Response

[
 {
    "panelID": 1,
    "trayID": 8,
    "plant": "Pouch 4a",
    "description": "New Product Trial",
    "status": "Prepare",
    "created": "2022-09-25T08:52:30"
  }
 ]

DELETE (Delete)

 Deletes all Users for specified Panel
 http://localhost:8080/c4jWS/Users?panelID={id}

 Deletes User as per parameters for panelID and userID
 http://localhost:8080/c4jWS/Users?panelID={id}&userID={id}
